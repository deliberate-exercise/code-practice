# 토이프로젝트 - 랜덤팀 생성

***

## 기능

### 1. 인증

- 로그인
- 로그아웃
- 회원가입
- 비밀번호 변경

### 2. 팀생성

#### (1) 랜덤팀(기본) : 총인원 - 제외인원

  - 총인원에서 제외인원 선별
  - 등록된 `user - pk(id)` 활용
  - 랜덤 처리
  - **홀수인원일 때도 처리** (ex. 7명 - A: 4 / B: 3 )
  - 회원이 아닌 게스트(용병) 도 추가하여 생성할 수 있도록 처리

#### (2) 밸런스 처리 : 선택인원 우선 팀 배정 - 제외인원 

  - 선택인원 팀에 우선배정 (A / B)
  - 제외인원 처리
  - 나머지 랜덤팀(기본) 기능 동일
